## Overview
- DISTORTED!!는 키보드를 들고 기타처럼 플레이하는 리듬 액션 게임입니다.
- 아케이드 게임 스타일의 UI 및 게임 플로우 감각을 재현하는 것을 목표로 클라이언트와 서버를 개발했습니다.
- 프로토타입을 학교 부스에서 시연해 약 300명의 이용자로부터 피드백을 얻을 수 있었습니다.
